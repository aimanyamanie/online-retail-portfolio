# Source data instructions

Dataset: UCI Online Retail
Download URL:
https://archive.ics.uci.edu/ml/machine-learning-databases/00352/Online%20Retail.xlsx

PowerShell download:
```powershell
curl.exe -L "https://archive.ics.uci.edu/ml/machine-learning-databases/00352/Online%20Retail.xlsx" -o data\online_retail.xlsx
```
